# Album Browser - Lab 6

## How to Run
1. Install Node.js from nodejs.org
2. Install Angular CLI: npm install -g @angular/cli
3. Install dependencies: npm install
4. Start: ng serve
5. Open: http://localhost:4200

## Routes
- /home - Welcome page
- /about - About page
- /albums - List of all albums
- /albums/:id - Album details with edit
- /albums/:id/photos - Album photos grid

## Tech
- Angular 21, HttpClient, JSONPlaceholder API
